E01 - Seguridad e Identidad - IAM

- HU 1.1 Autenticación al dashboard:
	Como Super Admin, quiero crear perfiles de administrador de e-commerce con permisos limitados, para delegar la operación de crear, administrar y eliminar descuentos, de temporada, fidelidad y tipo de producto del servicio sin comprometer la seguridad de la plataforma.

- HU 1.2 Gestión de credenciales: 
	Como plataforma e-commerce, quiero enviar mi API Key en cada petición, para autenticarme y consumir los servicios de loyalty de forma segura .

- HU 1.3 Diferencia de roles: 
	Como Super Admin, quiero poder revocar una API Key desde mi panel, para bloquear de inmediato el acceso de una integración si detecto una brecha de seguridad .
